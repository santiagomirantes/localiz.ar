## El problema

Tanto la API de direcciones argentinas oficial (**georef**) como las de terceros (**Google Maps**, **Mapbox**, etc.) suelen estar [desactualizadas](https://github.com/santiagomirantes/localiz.ar/blob/main/comparison/index.md), restringidas por un **rate-limiting** estricto o sujetas a **pagos**.

## La solución

**localiz.ar** presenta una **API liviana y actualizada** de direcciones argentinas, junto a una **librería Front-End** que facilita la creación y posterior verificación de **formularios**.

## Ventajas

- **open-source**

- construido con datos de **OpenStreetMap Contributors**, **Georef** y distintos sitios provinciales

- **100% gratuito y escalable**

- **fácil** de integrar

- abierto a sugerencias de **actualizaciones de la comunidad**
